# Sample ReactJS App to test the SDK

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app). This app is 
used to test the Cooee's Web SDK in a React website.

## Development

1. In the parent root directory, run `npm link`
2. In the `react-app`, first do `npm install @letscooee/web-sdk`
3. Then run `npm link @letscooee/web-sdk`
4. In a new tab run `npm tsc --watch`
5. In The `react-app`, run `npm start`
